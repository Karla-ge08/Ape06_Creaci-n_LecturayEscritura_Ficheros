## 🎯 Objetivos de la Práctica

* **General:** Conocer el manejo de arreglos y listas que permiten la manipulación de grandes cantidades de datos.
* **Específicos:**
  * Implementar operaciones CRUD completas utilizando estructuras secuenciales y dinámicas.
  * Desarrollar soluciones con asignación flexible de memoria mediante `ArrayList` y `LinkedList` en tiempo de ejecución.
  * Analizar y evaluar la optimización de recursos y comportamiento de desbordamiento entre arreglos primitivos y dinámicos.
